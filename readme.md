## Repo Quick Look
A small toy app develop with Github search repositories api based on Kotlin and MVVM architecture
#### Tech Stack, Libraries
1. Kotlin base
2. Coroutine + Flow + LiveData 
3. Jetpack 
	 - Lifecycle + ViewModel 
	 - ViewBinding
	 - Hilt
4. Retrofit2 + Moshi 
5. Junit + Hamcrest + Mockk + Espresso
